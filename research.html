DhakaRoadNet EdgeAI

An end-to-end road-object detection framework for complex Dhaka traffic.

2,145images
23,448bounding boxes
24classes
0.743test mAP50

Research status Manuscript submitted to COMPAS 26 · First Author

RepositoryDemo videoPlaylistAndroid APK
Show research detailsCollapse research details

Dataset

  • 1,754 training, 195 validation, and 196 test images.
  • Twenty-four classes representing heterogeneous Dhaka road scenes.
  • Prepared and verified in YOLOv8 annotation format.

Model and evaluation

  • YOLOv8n, 640 × 640 input, 120 epochs, batch size 16, AdamW.
  • 3,015,528 parameters.
  • Validation mAP50 0.798; test mAP50 0.743.
  • Weak-class analysis included.

Edge deployment

  • TensorFlow Lite FP16 export after model testing.
  • Runs without a server or API dependency.
  • Supports gallery, camera, live video, result saving, and annotation details.
