# Md. Shajibul Islam — Academic Portfolio

A lightweight academic portfolio focused on research, publications, teaching, selected engineering work, and preparation for Fall 2027 CS/AI PhD applications.

Live site: [https://shajibewucse19.github.io/portfolio/](https://shajibewucse19.github.io/portfolio/)

## Local preview

Open `index.html` directly, or serve the repository root with any static HTTP server.

## Deployment

Pushes to `main` trigger the GitHub Pages workflow in `.github/workflows/pages.yml`. The workflow publishes the repository root without a framework or build step.

## Privacy

The local `my_info` directory is intentionally ignored. Only selected information and assets copied into the public website are committed.
